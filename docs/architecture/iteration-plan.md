# Iteration Plan
## Post-MVP Release Timeline

### Phase 1: Core Improvements (1.x release)

#### High Priority

1. **Notification System Overhaul**
   - Redesign notification architecture for reliability/functionality
   - Add recurring notification options (daily, weekly, monthly)
   - Add snooze functionality for notifications

2. **Critical Bug Fixes**
   - Fix occasional crashes/stutters during AI tagging
   - Improve error handling during file operations

3. **Data Management**
    - Implement cloud backup and synchronization
    - Add export/import functionality

#### Medium Priority

1. **Enhanced Content Support**
   - Add Markdown support for text formatting
        - Implement bulleted and numbered lists
        - Add basic text styling (bold, italic, underline)


### Phase 2: Feature Expansion (2.x, 3.x releases)

#### Medium Priority

1. **Media Attachments**
   - Add image support for notes
   - Add voice note recording and playback


#### Low Priority

1. **Collaboration Features**
   - Implement note sharing
   - Add basic collaboration features
   - Create public/private note permissions
