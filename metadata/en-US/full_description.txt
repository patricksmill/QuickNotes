A simple note-taking application for Android devices that makes taking and organizing notes simpler, leveraging the power of AI (user-provided OpenAI API key required).

<b>QuickNotes Features:</b>

<b>Features:</b>

* Quickly add a new note
* Easily search notes by title, content, and tags
* Filter notes based on tags
* Auto-tag a note based on keywords, or by using an Open AI API response
* Delete a note by swiping
* Schedule note reminders
* Pin notes to the top of the list
